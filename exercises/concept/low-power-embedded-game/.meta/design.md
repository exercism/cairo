# Design

## Goal

Introduce the student to tuples and how to work with them.

## Learning objectives

- know how to create tuples
- know how to destructure tuples into new variables, e.g. `let (x, y) = a_point;`;

## Concepts

- tuples

## Prerequisites

- traits
